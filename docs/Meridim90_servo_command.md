# Meridim90 servo command　　
Meridim90のIndex[20]から[22],[24]...[78]まで入る各サーボに対するコマンドの初期案  
Index[21],[23]...[79] には具体的な値が入る。例）[20]:1, [21]:9000 → ID0番のサーボをトルクオンで90度に移動  
公式対応しているコマンドはまだ0,1のみ。  
以下の条件で改訂したい。  
- サーボパラメータの変更には、少なくとも通信1回分を消費する前提。  
- ESP32,Arduino,Teensyはいずれもリトルエンディアン方式。  
- 上位8bitはフラグ、下位8ビットは値として扱う予定。  
- 下位8bitでは目標角度値など共通できるものは共通とし、メーカー独自のコマンドは後半に割り当てる。  
- メーカー独自のコマンドはなるべく連番で割り当てたい。  
- 上位8bitのフラグについても、ニーズに合わせてシステム予約の内容を決定したい。  
- 各社メーカー向けサーボモジュールを新規に作成する場合には、暫定的に下記のリストに準じて作成する。
